# CLOPE — алгоритм кластеризации (C# реализация)

## Описание проекта

Данный проект представляет собой реализацию алгоритма **CLOPE (Clustering with sLOPE)** на языке C#.

Алгоритм предназначен для кластеризации категориальных транзакционных данных и часто используется в задачах анализа покупательских корзин.

## Принцип работы алгоритма

CLOPE — это жадный алгоритм кластеризации, который максимизирует функцию прибыли:

$$
Profit = \frac{S}{W^r}
$$

где:

- **S** — общее количество элементов в кластере  
- **W** — количество уникальных элементов  
- **r** — коэффициент “отталкивания”, регулирующий плотность кластеров  

---

## Формат входных данных

Алгоритм работает с транзакциями, где каждая транзакция — это набор категориальных элементов:

```csharp
new Transaction(new[] { "a", "b", "c" })
```

Пример набора данных:

```csharp
var data = new List<Transaction>
{
    new(new[] {"a","b"}),
    new(new[] {"a","b","c"}),
    new(new[] {"a","c","d"}),
    new(new[] {"d","e"}),
    new(new[] {"d","e","f"}),
    new(new[] {"a","b","e"}),
    new(new[] {"d","e","a"}),
};
```

---

## Этапы работы алгоритма

### 1. Инициализация

Каждая транзакция последовательно обрабатывается и:
- либо добавляется в существующий кластер  
- либо создаёт новый кластер  

---

### 2. Выбор кластера

Для каждой транзакции `t` вычисляется изменение функции качества:

$$
\Delta Profit = Profit_{after} - Profit_{before}
$$

Транзакция помещается в кластер, где значение $\Delta Profit$ максимально.

---

### 3. Итеративная оптимизация

После первичного распределения алгоритм:
- удаляет транзакции из кластеров  
- пробует переназначить их  
- повторяет процесс до стабилизации  

---

## Основные формулы

### Функция качества кластера:

$$
Profit = \frac{S}{W^r}
$$

### Изменение качества:

$$
\Delta = \frac{S_{new}(N+1)}{W_{new}^r} - \frac{S \cdot N}{W^r}
$$
